Welcome to my website, I will be demonstrating HTML5 principles and techniques, routing in Express

Website Functionality

This site consists of 3 static views (Home, About, HTML Demo) as well as 5 routes that supply data , including: All Employees, Employees by Department, A Single Employee By Employee Number, All Managers and All Departments.
Links to redirect to different pages to get the desired output.

  1. Link to Home Page'/'
  2. Link to get all Employee Data '/employees'
  3. Link to get all Departments Data'/departments'
  4. Link to get all Employee Data with respect to their Department '/employees?department=5'.
  5. Link to get all Employee Data with respect to their Employee Number '/employees/260'. 260 is the Employee Number value.
  6. Link to get all Managers Data '/managers'
  7. Link for the About me page '/about'
  8. Link for Html Demo page '/htmlDemo'

Short Description

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. This website contains data of 280 employees.
